# Samba Lab

## Objetivo
Laboratório para configuração de um servidor Samba utilizando AlmaLinux.

## Ambiente

- AlmaLinux 9
- Samba
- Cliente Linux
- Rede local

## Funcionalidades

- Compartilhamento de arquivos
- Controle de permissões
- Usuários locais
- Testes de acesso

## Comandos utilizados

```bash
dnf install samba samba-client
systemctl enable smb --now
